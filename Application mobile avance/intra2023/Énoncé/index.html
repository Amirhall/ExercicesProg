QUESTION 1 - Mise en page

Tu dois créer un projet appelé nom_pages avec 3 pages et les mises en page suivantes :

Ecran 1 (Am)

1 point On a le bon titre, et le carré jaune fait 222 par 222.

1 point La zone bleu est 6x plus haute que le vert, le rouge 3x plus haut et le orange 4 fois.

Ecran 2 (Stram)

1 point Le titre est bon et on a un bouton flottant avec une icone de flèche vers la droite (Icons.arrow_right_alt)

Ecran 3 (Gram)

1 point Il y a un carré jaune centré de 300 par 300, pas de tiroir et pas de appbar.

1 point On trouve le texte Gram et en dessous un bouton "Retour vers Am" fonctionnel le tout centré.

Navigation

1 point Le tiroir est présent sur les 2 premiers écrans (Am et Stram) avec le ☰, les liens fonctionnent et on retrouve TON prénom et TON matricule.


Clique sur les images pour agrandir

QUESTION 2 - Accès réseau

Tu dois créer un projet appelé nom_web_service

Web service:

Ton code doit accéder à un service web disponible en méthode GET sur l'URL ci-dessous. 

https://examen-intra-a23.azurewebsites.net/exam/artiste/recherche/{terme}
Exemple concret 
https://examen-intra-a23.azurewebsites.net/exam/artiste/recherche/jean

Le paramètre terme est entré via un champ texte.

Le service retourne une liste d'artiste trouvés selon un terme de recherche.

Les données retournées par le service doivent être affichées dans un ListView. Le nom de l'artiste doit être au moins visible.

Pointage

1 points Ton application utilise un champ texte pour récupérer le terme. Si tu n'y arrives pas tu peux envoyer une constante mais tu perds ce point.

2 points Récupérer les données retournées par le service dans l'application et les afficher via un print ou tout autre moyen.

2 points Récupérer les données retournées par le service dans l'application et les afficher via un ListView où on voit tous les champs de données. Si tu as réussi à afficher une ListView tu obtiens également les 2 points précédents.
