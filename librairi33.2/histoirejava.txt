Java est un langage de programmation de haut niveau orienté objet créé par James Gosling et 
Patrick Naughton, employés de Sun Microsystems, avec le soutien de Bill Joy (cofondateur de Sun 
Microsystems en 1982), présenté officiellement le 23 mai 1995 au SunWorld.

Un logiciel écrit en language Java a pour particularité d'être compilé vers un code intermédiaire formé de bytecodes 
qui peut être exécutée dans une machine virtuelle Java (JVM) en 
faisant abstraction du système d'exploitation.
